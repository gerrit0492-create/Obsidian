# Notes

Your personal notes area — plain markdown, editable from anywhere, readable by Claude.

## Add or edit notes from your iPhone
1. Install the **GitHub** app (App Store) and open the repo `gerrit0492-create/Obsidian`.
2. Open the `notes/` folder → tap a `.md` file to edit, or **Add file** to create one.
3. Make your changes → **Commit changes**. Done — it's saved to GitHub.

(You can also edit on the GitHub website in Safari — same thing.)

## Have Claude use your notes
In a Claude Code session on this repo, just ask — for example:
- "Summarize everything in `notes/`."
- "Read `notes/How I use Claude with my notes.md` and expand it."
- "Add a note in `notes/` about <topic>."

Claude reads and edits these files directly. It runs in the cloud off GitHub, so this works
from your phone — no Obsidian and no sync apps required.
