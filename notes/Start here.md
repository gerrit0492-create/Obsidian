# Start here

Welcome to your notes. A few ideas to get going:

- Jot anything down as a new `.md` file in this folder.
- Link notes by typing `[[` and picking a note (handy if you ever open them in Obsidian).
- Ask Claude to summarize, expand, or organize them.

See [[How I use Claude with my notes]] for the workflow we set up.
