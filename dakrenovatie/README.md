# Dakrenovatie — offerte-tracker

Kleine Streamlit-app om dakrenovatie-offertes te volgen, te vergelijken op
`€/m² incl. btw` en te toetsen aan de marktprijs. Vooringe vuld met de
offerte van Dakbedrijf Westermeer (Compiègnehof 11, Eindhoven).

## Starten
```bash
pip install -r requirements.txt
streamlit run app.py
```
Opent op http://localhost:8501.
