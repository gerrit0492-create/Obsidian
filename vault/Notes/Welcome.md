# Welcome to your vault

This is a starter vault. A few things to try:

- Browse the [[Daily Note]], [[Meeting Note]] and [[Project]] templates in `Templates/`.
- Point the core **Templates** plugin at the `Templates/` folder
  (Settings → Templates → Template folder location).
- Run the repo's `note-lint` tool against this vault to check for broken links and stats.

## Folders
- `Daily Notes/` — one note per day.
- `Projects/` — longer-running efforts.
- `Notes/` — everything else (you're reading one).
- `Templates/` — reusable note skeletons.
